# QIL Web UI Kit

**Surface:** Marketing / research website
**Design width:** 1280px
**Status:** v1.0

## Components
- `Header.jsx` — top navigation with logo, nav links, CTA
- `Hero.jsx` — full-width hero with display type, subhead, CTA
- `LabCard.jsx` — research/project card with tag, title, description, date
- `Footer.jsx` — site footer with links and tagline

## Screens in index.html
1. **Home** — Hero, featured projects grid, about strip
2. **Research** — filtered project listing
3. **Project detail** — single project view

## Notes
- No real backend; all data is static JS arrays
- Click-thru prototype — navigation between screens is simulated
- No assets provided; logos drawn in CSS/HTML from design tokens
